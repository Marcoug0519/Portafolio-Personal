# Portafolio Personal Marco Umaña Godínez C5K366

Este proyecto es la implementación de un portafolio personal estático utilizando HTML y Tailwind CSS

---

## Documentación del Código y Cosas Aprendidas

Este proyecto fue desarrollado con la asistencia de una inteligencia artificial (Gemini), La asistencia se centró en la creación de la estructura base del portafolio y en la guía para la implementación y publicación del sitio.

### Aprendizajes Clave Obtenidos:

1.  **Sintaxis y Estructura Web (HTML y CSS):**
    * **HTML Semántico:** Comprendí la importancia de usar etiquetas `<section id="...">`
    * **Tailwind CSS:** Aprendí a utilizar un *framework* de CSS para aplicar estilos complejos rápidamente mediante clases utilitarias (`flex`, `grid`, `bg-dark-surface`), lo que optimiza el tiempo de diseño.
2.  **Manejo de Enlaces y Rutas:**
    * **Rutas y Servidores:** Entendí por qué los archivos no se cargaban en GitHub Pages (debido a la sensibilidad de mayúsculas y minúsculas) y cómo depurar errores de rutas (`assets/mi-imagen.jpg`).
    * **Enlaces de Contacto:** Practiqué la implementación de enlaces específicos como `mailto:`, asegurando la funcionalidad de contacto.

La IA proporcionó la base, pero el trabajo de implementación, depuración de errores y la publicación final fue ejecutado directamente.
